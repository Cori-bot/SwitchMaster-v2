# Tech Stack - SwitchMaster

## Core & Desktop
- **Electron (39.2.7):** Framework principal pour l'application desktop.
- **TypeScript (5.9.3):** Langage principal pour le Main et le Renderer, assurant la sécurité du typage.
- **Node.js:** Environnement d'exécution pour le processus Main.

## Frontend (Renderer)
- **React (19.2.3):** Bibliothèque pour l'interface utilisateur.
- **Vite (7.3.0):** Outil de build et serveur de développement ultra-rapide.
- **Tailwind CSS (4.1.18):** Framework CSS utilitaire pour le stylisation.
- **Framer Motion (12.23.26):** Bibliothèque d'animations pour une interface fluide.
- **Lucide React:** Bibliothèque d'icônes.

## Outils & Utilitaires
- **pnpm:** Gestionnaire de paquets (Exclusif).
- **electron-builder (26.0.12):** Pour la compilation et la création d'installateurs (NSIS).
- **fs-extra:** Manipulation étendue du système de fichiers.
- **electron-log:** Gestion des journaux applicatifs.
- **electron-updater:** Gestion des mises à jour automatiques.

## Qualité & Tests
- **Vitest (4.0.16):** Framework de tests unitaires et d'intégration.
- **Prettier:** Formatage automatique du code.

## Scripts & Automatisation
- **PowerShell:** Scripts (.ps1) pour l'automatisation système sous Windows.
- **tsx:** Exécution directe de fichiers TypeScript pour les scripts de build.
